Three main features & User Flow

Step 1: User clicks on April on the Homepage

Step 2: User choose to add a new story today.

Step 3: User choose avatar as the story teller of today

Step 4: User choose the scene (AR background or pictures)

Step 5: User generate today's story by writing or recording.

Step 6: User choose to use voice record.

Step 7: User edit the story today.

Step 8: User post today's story.
